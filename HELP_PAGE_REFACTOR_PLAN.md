# 도움말 페이지 리팩토링 계획

## 현재 문제점 진단

### 1. 구조적 문제
```
현상: 섹션별 구분이 약하고 정보가 어지럽게 나열됨
- "빠른 도움"과 "응급 연락처"가 유사한 정보를 중복 표시
- 응급실 정보가 두 섹션에 모두 존재
- 섹션 간 시각적 구분이 부족
```

### 2. 디자인 일관성 부족
```
현상: 다른 페이지(카페, BBQ)와 스타일이 불일치
- 카페/BBQ는 배경색으로 섹션 구분 → 도움말은 단순 나열
- 히어로 섹션 없이 바로 정보 나열 시작
- 카드 디자인이 다른 페이지와 상이
```

### 3. 정보 과부하
```
현상: 한 화면에 너무 많은 정보
- 빠른 도움 / 응급 연락처 / FAQ / 안전 정보가 모두 노출
- FAQ 검색/필터가 복잡해 보임
- 안전 정보 섹션이 다소 뜬금없음
```

---

## 리팩토링 목표

1. **정보 구조화** - 중복 제거, 핵심 기능 우선
2. **시각적 구분** - 섹션별 배경/테두리로 명확한 분리
3. **디자인 통일** - 카페/BBQ 페이지와 일관된 스타일
4. **사용성 개선** - 긴급 상황 시 빠른 접근

## 제거/이동 항목

### 제거
| 항목 | 이유 |
|------|------|
| FAQ 검색창 | 복잡도 증가, 실사용 적음 |
| 카테고리 필터 | FAQ 수가 적어 불필요 |
| 안전 정보 섹션 | 가이드 페이지로 이동 권장 |
| WiFi 바로가기 | 비밀번호 없음 |
| 24시간 약국 | 해당 없음 → 편의점으로 대체 |
| 빠른 도움 섹션 | 응급 연락처와 중복, 제거 |

### 이동
| 항목 | 현재 위치 | 이동 위치 |
|------|----------|----------|
| 관리자 전화 | 빠른 도움 (상단) | 페이지 하단 (위계 낮춤) |
| 응급실 | 빠른 도움 + 응급 연락처 | 응급 연락처만 |
