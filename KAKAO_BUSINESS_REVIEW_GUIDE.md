**현재 상태:**

## 📸 서비스 화면 준비 가이드

### 서비스 화면이 필요한 이유

카카오는 실제 서비스가 운영 중인지 확인하기 위해 서비스 화면을 요구합니다. 특히:
- 오픈 전 서비스
- 회원가입/로그인이 필요한 서비스
- 웹 또는 앱 URL로 서비스 확인이 어려운 경우

## 🎯 추천 서비스 화면 구성

### 옵션 1: 웹 앱 주요 화면 (권장)

다음 화면들을 하나의 이미지로 합성하거나, 각각 스크린샷을 준비:

#### 1. 홈 화면 (HOME)
- 게스트 환영 메시지
- WiFi 정보
- 체크인/체크아웃 시간
- 일몰 시간 위젯
- 주문 내역

**캡처 방법:**
- 웹 브라우저에서 `https://ouscaravan.com/home?guest=테스트&room=A1` 접속
- 스크린샷 캡처

#### 2. 가이드 화면 (GUIDE)
- 디지털 매뉴얼
- 검색 기능
- 카테고리 필터
- BBQ 가이드 캐러셀

**캡처 방법:**
- `/guide` 페이지 접속
- 스크린샷 캡처

#### 3. 마켓 화면 (MARKET)
- 디지털 쿠폰 (3D 플립 애니메이션)
- 카페 메뉴 캐러셀
- 불멍/바베큐 주문 폼

**캡처 방법:**
- `/market` 페이지 접속
- 스크린샷 캡처

#### 4. 도움말 화면 (HELP)
- 응급 연락처
- 안전 정보
- FAQ

**캡처 방법:**
- `/help` 페이지 접속
- 스크린샷 캡처

### 방법 1: 실제 웹 앱 스크린샷 (가장 권장)

#### 준비할 화면:

1. **홈 화면 (메인 대시보드)**
   - URL: `https://ouscaravan.com/home?guest=테스트게스트&room=A1&checkin=2026-01-15&checkout=2026-01-17`

     - 개인화된 환영 메시지

2. **가이드 화면**
   - URL: `https://ouscaravan.com/guide`
   - 캡처 내용:
     - 검색 기능
     - 가이드 목록

3. **마켓 화면**
   - URL: `https://ouscaravan.com/market`

     - 디지털 쿠폰
     - 메뉴 캐러셀
     - 주문 폼

4. **카카오톡 메시지 수신 화면**

**옵션 A: 4개 화면을 하나로 합성**
- 상단: 홈 화면

- 하단: 카카오톡 메시지 화면

**옵션 B: 가장 대표적인 화면 1개만 제출**
- 홈 화면 (메인 대시보드)만 제출

2. **주요 기능**

## 💡 추천 제작 방법

### 가장 간단한 방법:

1. **웹 앱 홈 화면 캡처**
   - `https://ouscaravan.com/home?guest=테스트&room=A1` 접속
   - 브라우저 전체 화면 캡처 (F12 → 디바이스 모드 → 모바일 뷰)
   - 스크린샷 저장

2. **카카오톡 메시지 화면 캡처**
   - 테스트 메시지 발송
   - 카카오톡 앱에서 메시지 수신 화면 캡처

3. **이미지 합성**
   - 두 화면을 하나의 이미지로 합성
   - 상단: 웹 앱 홈 화면
   - 하단: 카카오톡 메시지 화면

### 더 상세한 방법:

1. **주요 화면 4개 캡처**
   - 홈, 가이드, 마켓, 도움말

2. **이미지 편집 도구로 합성**
   - 2x2 그리드로 배치
   - 각 화면에 라벨 추가

3. **서비스 설명 텍스트 추가**
   - 상단에 서비스 이름
   - 각 화면에 간단한 설명
