# 알림톡 메뉴 위치 찾기 가이드

## 📍 현재 화면 분석

카카오 비즈니스 파트너센터 대시보드에서 알림톡 메뉴를 찾는 방법입니다.

---

## 🔍 알림톡 메뉴 위치

### 방법 1: "메시지" 메뉴 확인

**현재 화면에서:**

1. 왼쪽 사이드바에서 **"메시지"** 메뉴 클릭
   - 아이콘: 말풍선에 플러스(+) 표시
   - 위치: "채팅" 메뉴 아래

2. **"메시지"** 메뉴 클릭 후:
   - 드롭다운 메뉴 또는 서브 메뉴 확인
   - **"알림톡"** 또는 **"알림톡 관리"** 메뉴 찾기

### 방법 2: 상단 네비게이션 확인

**상단 메뉴바에서:**

1. **"비즈도구"** 메뉴 클릭
2. 드롭다운에서 **"알림톡"** 또는 **"메시지 발송"** 찾기

### 방법 3: 직접 URL 접속

**알림톡 관리 페이지 직접 접속:**

```
https://business.kakao.com/alimtalk
```

또는

```
https://business.kakao.com/message/alimtalk
```

## ⚠️ 알림톡 메뉴가 보이지 않는 경우

### 원인 1: 알림톡 서비스 미신청

**해결 방법:**

1. **"메시지"** 메뉴 클릭
2. **"서비스 신청"** 또는 **"알림톡 시작하기"** 버튼 찾기
3. 서비스 신청 진행

### 원인 2: 채널 인증 미완료

**확인 사항:**

1. 왼쪽 사이드바 상단의 **"오우스마켓"** 프로필 클릭
2. 채널 상태 확인
3. 인증 완료 여부 확인

**인증 미완료 시:**
- 채널 인증 절차 완료 필요
- 인증 완료 후 알림톡 메뉴 표시

### 원인 3: 권한 부족

**확인 사항:**

1. 현재 로그인한 계정의 권한 확인
2. 채널 관리자 권한이 있는지 확인

**해결 방법:**
- 채널 관리자에게 권한 요청
- 또는 관리자 계정으로 로그인

## 📋 단계별 확인 방법

### 1단계: 메시지 메뉴 확인

**현재 화면에서:**

1. 왼쪽 사이드바의 **"메시지"** 메뉴 확인
   - 아이콘: 말풍선에 플러스(+)
   - 위치: "채팅" 아래

2. **"메시지"** 메뉴 클릭

3. 다음 중 하나 확인:
   - **"알림톡"** 서브 메뉴
   - **"알림톡 관리"** 메뉴
   - **"서비스 신청"** 버튼

### 2단계: 비즈도구 메뉴 확인

**상단 네비게이션에서:**

1. **"비즈도구"** 메뉴 클릭
2. 드롭다운 메뉴에서 **"알림톡"** 찾기

### 3단계: 직접 검색

**파트너센터 내 검색:**

1. 상단 검색창에 **"알림톡"** 입력
2. 검색 결과에서 알림톡 관련 메뉴 찾기

---

## 🎯 알림톡 서비스 신청 방법

### 방법 1: 메시지 메뉴에서 신청

1. **"메시지"** 메뉴 클릭
2. **"알림톡 시작하기"** 또는 **"서비스 신청"** 버튼 클릭
3. 서비스 약관 동의
4. 신청 완료

### 방법 2: 고객센터 문의

1. 상단 메뉴에서 **"고객센터"** 클릭
2. **"알림톡 서비스 신청"** 문의
3. 신청 절차 안내 받기

### 방법 3: 카카오 비즈니스 홈페이지

1. [카카오 비즈니스](https://business.kakao.com/) 접속
2. **"서비스"** → **"알림톡"** 메뉴 확인
3. 서비스 신청 진행

2. 고객센터 문의
3. 카카오 비즈니스 홈페이지에서 직접 신청
