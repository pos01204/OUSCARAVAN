# 카카오톡 채널 마케팅 전략

## 📋 전략 개요

### 목표
- 알림톡으로 예약 완료 즉시 발송
- 채널 친구 추가 유도
- 친구톡을 통한 지속적인 마케팅

### 전략 흐름
```
[예약 완료]
  ↓
[알림톡 발송] (즉시)
  ↓
[채널 친구 추가 유도]
  ↓
[친구 추가 완료]
  ↓
[친구톡 마케팅 발송]
  - 프로모션
  - 시즌별 혜택
  - 재방문 유도
  - 리뷰 요청
```

---

## 1단계: 채널 친구 추가 유도

### 1-1. 알림톡 메시지에 친구 추가 혜택 포함

**예약 완료 알림톡 메시지:**
```
#{guest_name}님, OUSCARAVAN 예약이 완료되었습니다!

📅 예약번호: #{reservation_number}
📅 체크인: #{checkin_date} 15:00
📅 체크아웃: #{checkout_date} 11:00
🏠 객실: #{room_type}
💰 결제금액: #{amount}원

컨시어지 서비스를 이용하시려면 아래 버튼을 클릭하세요.

💡 OUSCARAVAN과 친구가 되시면:
✓ 체크인/체크아웃 안내 메시지 수신
✓ 특별 혜택 및 프로모션 알림
✓ 빠른 고객 지원
```

**버튼 구성:**
1. **링크 버튼**: "컨시어지 서비스 이용하기"
2. **채널 추가 버튼**: "친구 추가하고 혜택받기" (선택사항)

### 1-2. 별도 친구 추가 유도 메시지

**발송 시점:**
- 예약 완료 후 1-2시간 후
- 또는 체크인 당일 오전

**템플릿 내용:**
```
#{guest_name}님, OUSCARAVAN과 친구가 되어주세요!

친구 추가 시 받을 수 있는 혜택:
🎁 체크인/체크아웃 자동 안내
🎁 시즌별 특별 프로모션 알림
🎁 생일 축하 쿠폰 제공
🎁 재방문 할인 혜택
🎁 빠른 고객 지원

지금 친구 추가하고 다양한 혜택을 받아보세요!
```

**버튼:**
- 채널 추가 버튼

## 2단계: 친구 목록 관리

### 2-1. 친구 목록 조회 및 저장

**n8n 워크플로우:**
```
Cron Trigger (매일 오전 9시)
  ↓
HTTP Request (친구 목록 조회)
  ↓
Code (데이터 정리)
  ↓
데이터베이스 저장 (또는 Google Sheets)
```

**친구 목록 조회 API:**
```
GET https://kapi.kakao.com/v1/api/talk/friends
```

**저장할 데이터:**
- 친구 UUID
- 친구 닉네임
- 프로필 이미지
- 친구 추가 일시
- 마지막 예약 정보

### 2-2. 친구 추가 추적

**시스템:**
- 예약 완료 시 게스트 정보 저장
- 친구 목록 조회 시 매칭
- 친구 추가 여부 추적

## 3단계: 친구톡 마케팅 전략

### 3-1. 마케팅 메시지 유형

#### 유형 1: 프로모션 안내
```
#{guest_name}님, OUSCARAVAN 특별 프로모션!

이번 주말 특가 예약 진행 중입니다.
기존 고객님께는 추가 10% 할인 혜택을 드립니다.

예약하기: #{booking_link}
```

#### 유형 2: 시즌별 혜택
```
#{guest_name}님, 봄 시즌 특별 혜택!

봄 캠핑 시즌을 맞아 특별 패키지를 준비했습니다.
불멍 세트 무료 제공 + 카페 음료 2잔 서비스

예약하기: #{booking_link}
```

#### 유형 3: 재방문 유도
```
#{guest_name}님, 다시 만나요!

지난번 OUSCARAVAN에서 즐거운 시간 보내셨나요?
재방문 고객님께는 특별 할인 혜택을 드립니다.

예약하기: #{booking_link}
```

#### 유형 4: 리뷰 요청
```
#{guest_name}님, OUSCARAVAN은 어떠셨나요?

소중한 후기를 남겨주시면 다음 예약 시
10% 할인 쿠폰을 드립니다.

리뷰 작성하기: #{review_link}
```

#### 유형 5: 생일 축하
```
#{guest_name}님, 생일 축하합니다! 🎉

생일을 맞아 특별 선물을 준비했습니다.
이번 달 예약 시 생일 케이크 서비스!

예약하기: #{booking_link}
```

### 3-2. 마케팅 발송 시점

**일정:**
- **월요일 오전**: 주간 프로모션 안내
- **시즌 시작 전**: 시즌별 특별 혜택
- **체크아웃 후 1일**: 리뷰 요청
- **생일 당일**: 생일 축하 메시지
- **재방문 유도**: 마지막 방문 후 1개월

## 4단계: n8n 마케팅 워크플로우

### 4-1. 주간 프로모션 발송

**워크플로우:**
```
Cron Trigger (매주 월요일 오전 9시)
  ↓
친구 목록 조회
  ↓
프로모션 메시지 템플릿 선택
  ↓
친구톡 발송
```

**Code 노드 (프로모션 메시지 생성):**
```javascript
const friends = $input.all()[0].json.elements;
const promotionLink = 'https://booking.naver.com/...';

const messages = friends.map(friend => {
  const templateObject = {
    object_type: "text",
    text: `${friend.profile_nickname}님, OUSCARAVAN 특별 프로모션!\n\n이번 주말 특가 예약 진행 중입니다.\n기존 고객님께는 추가 10% 할인 혜택을 드립니다.`,
    link: {
      web_url: promotionLink,
      mobile_web_url: promotionLink
    },
    button_title: "예약하기"
  };

  return {
    json: {
      friend_uuid: friend.uuid,
      friend_nickname: friend.profile_nickname,
      template_object: JSON.stringify(templateObject)
    }
  };
});

return messages;
```

### 4-2. 리뷰 요청 발송

**워크플로우:**
```
Cron Trigger (매일 오전 10시)
  ↓
체크아웃 완료 고객 조회 (1일 전)
  ↓
친구 목록에서 매칭
  ↓
친구톡 발송 (리뷰 요청)
```

### 4-3. 생일 축하 발송

**워크플로우:**
```
Cron Trigger (매일 오전 9시)
  ↓
오늘 생일인 고객 조회
  ↓
친구 목록에서 매칭
  ↓
친구톡 발송 (생일 축하)
```

## 📋 마케팅 일정표

### 주간 일정
- **월요일**: 주간 프로모션 안내
- **수요일**: 시즌별 특별 혜택
- **금요일**: 주말 예약 유도

### 월간 일정
- **월초**: 월간 프로모션 안내
- **월중**: 시즌별 특별 이벤트
- **월말**: 다음 달 예약 유도

### 시즌별 일정
- **봄 시즌**: 봄 캠핑 프로모션
- **여름 시즌**: 여름 특별 패키지
- **가을 시즌**: 가을 단풍 프로모션
- **겨울 시즌**: 겨울 특별 혜택
