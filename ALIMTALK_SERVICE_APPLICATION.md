# 알림톡 서비스 신청 가이드

## 📋 현재 상황

**현재 화면:** "메시지 보내기" (채널 메시지/친구톡)  
**찾는 기능:** 알림톡 서비스 신청

**중요:** 알림톡은 "메시지 보내기"와는 **별도의 서비스**입니다. 알림톡은 카카오 비즈니스에서 별도로 신청해야 합니다.

---

## 🔍 알림톡과 채널 메시지의 차이

### 채널 메시지 (현재 화면)
- **위치:** 메시지 → 메시지 보내기
- **특징:** 친구로 추가된 고객에게만 발송 가능
- **비용:** 무료 (일일 한도 존재)
- **용도:** 마케팅, 공지사항

### 알림톡 (별도 서비스)
- **위치:** 별도 메뉴 또는 카카오 비즈니스 홈페이지
- **특징:** 친구 추가 없이 전화번호로 발송 가능
- **비용:** 유료 (건당 20-30원)
- **용도:** 예약 안내, 주문 확인, 알림 등

---

## 🎯 알림톡 서비스 신청 방법

### 방법 1: 카카오 비즈니스 홈페이지에서 직접 신청 (권장)

1. **새 탭 열기**
   - 브라우저에서 새 탭 열기
   - 또는 현재 탭에서 이동

2. **카카오 비즈니스 홈페이지 접속**
   ```
   https://business.kakao.com/
   ```

3. **"서비스" 메뉴 확인**
   - 상단 메뉴에서 **"서비스"** 또는 **"알림톡"** 찾기
   - 또는 **"비즈니스 도구"** → **"알림톡"** 클릭

4. **알림톡 서비스 신청**
   - **"알림톡 시작하기"** 또는 **"서비스 신청"** 버튼 클릭
   - 서비스 약관 동의
   - 신청 완료

### 방법 2: 고객센터 문의

1. **현재 화면에서:**
   - 상단 메뉴의 **"고객센터"** 클릭

2. **문의하기:**
   - **"1:1 문의"** 또는 **"상담 신청"** 클릭
   - 문의 내용: "알림톡 서비스 신청 방법 문의"
   - 신청 절차 안내 받기

### 방법 3: 카카오 비즈니스 콘솔에서 확인

1. **현재 화면에서:**
   - 상단 메뉴의 **"비즈도구"** 클릭
   - 드롭다운에서 **"알림톡"** 또는 **"메시지 발송"** 찾기

2. **또는:**
   - 왼쪽 사이드바에서 다른 메뉴 확인
   - **"인사이트"** 또는 **"홍보"** 메뉴에서 알림톡 관련 링크 찾기

   - 채널 설정에서 인증 상태 확인

2. **사업자 정보 확인:**
   - 채널 설정에서 사업자 정보 확인

## 🔧 단계별 신청 가이드

### 1단계: 카카오 비즈니스 홈페이지 접속

1. 새 탭 열기
2. `https://business.kakao.com/` 접속
3. 로그인 (필요 시)

### 2단계: 알림톡 서비스 페이지 찾기

**옵션 A: 상단 메뉴에서**
- **"서비스"** → **"알림톡"** 클릭

**옵션 B: 직접 검색**
- 홈페이지에서 "알림톡" 검색
- 검색 결과에서 알림톡 서비스 페이지 클릭

**옵션 C: URL 직접 접속**
- `https://business.kakao.com/services/alimtalk` 접속

### 3단계: 서비스 신청

1. **"알림톡 시작하기"** 또는 **"서비스 신청"** 버튼 클릭
2. 서비스 약관 동의
3. 채널 선택 (오우스마켓)
4. 신청 완료

### 4단계: 승인 대기

- 서비스 승인까지 **1-2일** 소요
- 승인 완료 후 알림톡 메뉴 표시
- 템플릿 등록 가능

## 💡 현재 화면에서 할 수 있는 것

### 채널 메시지 활용 (임시 방안)

알림톡 승인 전까지는 **채널 메시지(친구톡)**를 활용할 수 있습니다:

1. **"메시지 보내기"** 페이지에서:
   - 친구로 추가된 고객에게 메시지 발송
   - 예약 안내 메시지 작성
   - 템플릿 활용

2. **제한사항:**
   - 친구로 추가된 고객에게만 발송 가능
   - 일일 발송 한도 존재

## 🆘 문제 해결

### 문제 1: 알림톡 메뉴가 전혀 보이지 않음

**원인:** 서비스를 아직 신청하지 않음

**해결:**
1. 카카오 비즈니스 홈페이지에서 직접 신청
2. 또는 고객센터 문의

### 문제 2: 서비스 신청 버튼을 찾을 수 없음

**해결:**
1. `https://business.kakao.com/services/alimtalk` 직접 접속
2. 고객센터 문의

### 문제 3: 채널 인증 미완료

**확인:**
1. 채널 설정에서 인증 상태 확인
2. 인증 미완료 시 인증 절차 진행
