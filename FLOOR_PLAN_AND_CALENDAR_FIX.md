# 약도 7~10호 배치 및 캘린더 높이 수정 완료

## ✅ 완료된 작업

### 1. 7~10호를 ㅜ 형태로 수정

**파일**: `lib/constants/floorPlan.ts`

**문제점**:
- 기존에 7-10호가 2x2 형태(ㅁ)로 배치되어 있었음
- 실제 약도 구조는 ㅜ 형태여야 함 (상단: 7,8,9호, 하단: 10호)

**수정 내용**:
- 상단 행: 7호, 8호, 9호 (3개 가로 배치)
- 하단 행: 10호 (1개만, 중앙 정렬)
- 사용 가능 너비: 600 - 436 = 164
- 각 방 너비: (164 - 6*2 간격) / 3 = 약 50

**새로운 배치**:
```
[건물/창고]  [7호] [8호] [9호]
[주차공간 3]      [10호]
```

**정확한 좌표**:
- 7호: `x: 436, y: 130, width: 50`
- 8호: `x: 492, y: 130, width: 50`
- 9호: `x: 548, y: 130, width: 50`
- 10호: `x: 492, y: 195, width: 50` (중앙 정렬)
- 최대 x 좌표: 548 + 50 = 598 (viewBox 600 내에 맞춤)

### 2. 관리자 페이지 캘린더 높이 조정

**파일**: `app/admin/reservations/ReservationCalendarView.tsx`, `app/globals.css`

**문제점**:
- 캘린더가 중간 정도에서 아래 날짜가 잘리고 있었음
- `h-[calc(100vh-280px)]`가 충분하지 않았음

**수정 내용**:

1. **컨테이너 높이 증가** (`ReservationCalendarView.tsx`):
   - 모바일: `h-[calc(100vh-280px)]` → `h-[calc(100vh-200px)]` (80px 증가)
   - 데스크톱: `md:h-[600px]` → `md:h-[700px]` (100px 증가)

2. **월간 뷰 최소 높이 보장** (`app/globals.css`):
   - `.rbc-month-view`에 `min-height: 600px !important;` 추가
   - `.rbc-month-row`에 `min-height: 200px !important;` 추가
