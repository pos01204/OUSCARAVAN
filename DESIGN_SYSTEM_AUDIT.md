- `accent`: `#FF7E5F` (primary와 동일)
- `border`: `#221E1D` (foreground와 동일)

**카드 색상:**
- `card`: `#FFFFFF` (흰색)

### 2. 타이포그래피

#### 폰트 설정

**본문 폰트:**
- `Pretendard` (한글 최적화 폰트)
- CDN을 통한 로드: `https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.min.css`
- 폴백: `-apple-system, BlinkMacSystemFont, system-ui, sans-serif`

**제목 폰트:**
- `Montserrat` (영문 폰트)
- Google Fonts를 통한 로드
- `tracking-wide` 적용 (자간 넓게)

**상태**: ✅ 일관된 타이포그래피 시스템

#### 폰트 크기

Tailwind CSS 기본 크기 사용:
- `text-xs`: 0.75rem (12px)
- `text-sm`: 0.875rem (14px)
- `text-base`: 1rem (16px)
- `text-lg`: 1.125rem (18px)
- `text-xl`: 1.25rem (20px)
- `text-2xl`: 1.5rem (24px)
- `text-3xl`: 1.875rem (30px)

**상태**: ✅ 일관된 폰트 크기 시스템

### 3. 간격 및 레이아웃

#### 컨테이너 설정

**기본 컨테이너:**
- `container`: 중앙 정렬, 패딩 2rem
- 최대 너비: `2xl` (1400px)

**페이지별 컨테이너:**
- 관리자 페이지: `container mx-auto px-4 py-8`
- 고객 페이지: `container mx-auto max-w-md px-4 py-6 md:max-w-2xl`

**상태**: ✅ 일관된 레이아웃 시스템

#### 간격 시스템

Tailwind CSS 기본 간격 사용:
- `gap-2`: 0.5rem (8px)
- `gap-4`: 1rem (16px)
- `gap-6`: 1.5rem (24px)
- `space-y-4`: 세로 간격 1rem

**상태**: ✅ 일관된 간격 시스템

### 4. 아이콘 일관성

#### 사용 중인 아이콘 라이브러리

**Lucide React:**
- 모든 페이지에서 일관되게 사용
- 예: `Search`, `Menu`, `X`, `RefreshCw`, `Loader2`, `Clock`, `Phone`

**상태**: ✅ 단일 아이콘 라이브러리 사용

#### 아이콘 크기

일관된 크기 사용:
- 작은 아이콘: `h-4 w-4` (16px)
- 중간 아이콘: `h-5 w-5` (20px)
- 큰 아이콘: `h-6 w-6` (24px)
- 매우 큰 아이콘: `h-8 w-8` (32px)

**상태**: ✅ 일관된 아이콘 크기

## 📱 반응형 디자인 현황

### 모바일 레이아웃

**브레이크포인트:**
- 기본: 모바일 우선
- `md:` (768px 이상): 태블릿/데스크톱

**관리자 페이지:**
- 모바일: 햄버거 메뉴 (`AdminMobileNav`)
- 데스크톱: 가로 네비게이션 (`hidden md:flex`)

**고객 페이지:**
- 모바일: 하단 네비게이션 (`pb-16 md:pb-0`)
- 데스크톱: 하단 네비게이션 숨김

**상태**: ✅ 모든 페이지 반응형 대응

### 태블릿 레이아웃

**그리드 레이아웃:**
- `md:grid-cols-2`: 태블릿에서 2열
- `lg:grid-cols-3`: 데스크톱에서 3열
- `lg:grid-cols-4`: 데스크톱에서 4열
- `lg:grid-cols-5`: 데스크톱에서 5열

**상태**: ✅ 태블릿 레이아웃 최적화

### 데스크톱 레이아웃

**최대 너비:**
- 관리자 페이지: `container` (1400px)
- 고객 페이지: `max-w-2xl` (672px)

**상태**: ✅ 데스크톱 레이아웃 최적화
