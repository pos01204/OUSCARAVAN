## 1. 현재 구조 분석

### 1.1 컴포넌트 구성 (순서대로)

```
┌─────────────────────────────────────┐
│         GuestPageHeader             │  ← "이용 안내서" 타이틀
├─────────────────────────────────────┤
│       Category Filter Buttons       │  ← 전체/시설/체크인/... 버튼
├─────────────────────────────────────┤
│         Search Input                │  ← 검색 필드
├─────────────────────────────────────┤
│      BBQ Guide Toggle Card          │  ← 불멍/바베큐 가이드 진입
```

### 1.2 관련 컴포넌트

| 컴포넌트 | 역할 | 위치 |
|----------|------|------|
| `GuestGuideContent` | 메인 컨테이너 | guest/ |
| `StepByStepGuide` | 단계별 가이드 (페이저) | features/ |
| `GuideChecklist` | 체크리스트 | features/ |
| `GuideFAQ` | FAQ 아코디언/페이저 | features/ |
| `GuideTroubleshooting` | 문제 해결 가이드 | features/ |
| `TrashCategoryGuide` | 쓰레기 분류 가이드 | features/ |
| `BBQCarousel` | BBQ 캐러셀 | features/ |

### 1.3 발견된 문제점

| 영역 | 문제 | 심각도 |
|------|------|--------|
| **정보 밀도** | BBQ 카드 내 요약+배지+버튼 과다 | 🔴 높음 |
| **카테고리 필터** | 버튼 스타일이 플랫하고 선택 상태 불명확 | 🟡 중간 |
| **검색 UX** | 검색 필드 위치가 카테고리 아래로 접근성 낮음 | 🟡 중간 |
| **가이드 카드** | 클릭 가능 힌트 부족, 카테고리 배지 위치 어색 | 🟡 중간 |
| **상세 탭** | 탭 레이블 너무 짧음(단계별→Steps), 의미 전달 부족 | 🟡 중간 |
| **빈 상태** | 검색 결과 없을 때 디자인 빈약 | 🟢 낮음 |
| **아이콘 일관성** | 경고/팁/체크 아이콘 크기/색상 불통일 | 🟢 낮음 |
| **모바일 스크롤** | 카테고리 필터 가로 스크롤 힌트 없음 | 🟢 낮음 |

## 2. 리팩토링 목표

### 2.1 핵심 원칙

```
✓ 정보 계층화 — 중요한 정보를 시각적으로 강조
✓ 접근성 향상 — 검색/필터 사용성 개선
✓ 시각적 통일 — 카드/버튼/아이콘 일관된 스타일
✓ 모바일 최적화 — 터치 친화적 UI
```

### 2.2 기대 효과

- **탐색 시간 단축**: 카테고리/검색 접근성 향상
- **정보 가독성 향상**: 카드 디자인 정리로 스캔 용이
- **사용 만족도 향상**: 인터랙션 피드백 강화

### 3.2 카테고리 필터 개선

**현재:**
```
[전체] [시설안내] [체크인] [체크아웃] [기타]
  ↑ 플랫 버튼, 선택 상태 불명확
```

**변경안:**
```
┌─────────────────────────────────────────────────┐
│  🏷️ ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐   │
│     │ 전체  │ │ 시설  │ │체크인 │ │ BBQ  │ ▸ │
│     │  ●    │ │       │ │       │ │       │   │
│     └───────┘ └───────┘ └───────┘ └───────┘   │
│       선택됨    칩 스타일, 우측 스크롤 힌트     │
└─────────────────────────────────────────────────┘
```

- **칩(Chip) 스타일**: pill 형태, 선택 시 배경 강조
- **선택 인디케이터**: 하단 도트 또는 배경 색상 변경
- **스크롤 힌트**: 우측 페이드 그라데이션 + 화살표

### 4.2 카테고리 칩 컴포넌트

```tsx
// GuideCategoryChips.tsx
interface GuideCategoryChipsProps {
  categories: string[];
  selected: string;
  onSelect: (category: string) => void;
}

export function GuideCategoryChips({ 
  categories, 
  selected, 
  onSelect 
}: GuideCategoryChipsProps) {
  return (
    <div className="relative">
      <div className="flex gap-2 overflow-x-auto pb-2 scrollbar-hide">
        {categories.map((cat) => (
          <button
            key={cat}
            onClick={() => onSelect(cat)}
            className={cn(
              "shrink-0 px-4 py-2 rounded-full text-sm font-medium transition-all",
              selected === cat
                ? "bg-brand-dark text-white"
                : "bg-background-muted text-muted-foreground hover:bg-background-accent"
            )}
          >
            {cat}
          </button>
        ))}
      </div>
      {/* 우측 스크롤 힌트 */}
      <div className="absolute right-0 top-0 bottom-2 w-8 bg-gradient-to-l from-background to-transparent pointer-events-none" />
    </div>
  );
}
```

### 4.3 가이드 카드 컴포넌트

```tsx
// GuideCard.tsx
interface GuideCardProps {
  title: string;
  overview?: string;
  category: string;
  icon: LucideIcon;
  hasSteps?: boolean;
  hasChecklist?: boolean;
  hasFaq?: boolean;
  warning?: boolean;
  onClick: () => void;
}

export function GuideCard({ 
  title, 
  overview, 
  category, 
  icon: Icon,
  hasSteps,
  hasChecklist,
  hasFaq,
  warning,
  onClick 
}: GuideCardProps) {
  const contentTypes = [
    hasSteps && '단계별',
    hasChecklist && '체크리스트',
    hasFaq && 'FAQ',
  ].filter(Boolean).join(' · ');

  return (
    <Card 
      interactive 
      onClick={onClick}
      className="group cursor-pointer card-hover-glow"
    >
      <CardContent className="p-4">
        <div className="flex items-start gap-3">
          {/* 아이콘 배지 */}
          <CardIconBadge icon={Icon} tone={warning ? "warning" : "info"} />
          
          <div className="flex-1 min-w-0">
            {/* 타이틀 */}
            <h3 className="font-bold text-brand-dark truncate">{title}</h3>
            
            {/* 설명 */}
            {overview && (
              <p className="text-sm text-muted-foreground mt-1 line-clamp-2">
                {overview}
              </p>
            )}
            
            {/* 메타 정보 */}
            <div className="flex items-center gap-2 mt-2 text-xs text-muted-foreground">
              <span>{category}</span>
              {contentTypes && (
                <>
                  <span>·</span>
                  <span>{contentTypes}</span>
                </>
              )}
            </div>
          </div>
          
          {/* 화살표 */}
          <ChevronRight className="h-5 w-5 text-muted-foreground group-hover:translate-x-1 transition-transform shrink-0" />
        </div>
      </CardContent>
    </Card>
  );
}
```

## 5. 세부 변경 사항

### 5.1 검색 필드

| 항목 | 현재 | 변경 |
|------|------|------|
| **위치** | 카테고리 아래 | 카테고리 위 (최상단) |
| **플레이스홀더** | "안내 내용을 검색하세요..." | "검색어를 입력하세요" |
| **스타일** | 기본 Input | 둥근 모서리 + 그림자 |
| **클리어 버튼** | 없음 | 검색어 있을 때 X 버튼 |

### 5.2 카테고리 필터

| 항목 | 현재 | 변경 |
|------|------|------|
| **스타일** | outline/default 버튼 | pill 칩 |
| **선택 표시** | variant 변경 | 배경색 + 텍스트색 변경 |
| **스크롤** | 기본 overflow | 우측 페이드 힌트 |
| **간격** | gap-2 | gap-2 유지, 패딩 조정 |

### 5.3 BBQ 섹션

| 항목 | 현재 | 변경 |
|------|------|------|
| **레이아웃** | 내부 카드 + 배지 다수 | 단순 카드 1개 |
| **정보 밀도** | 높음 (요약+배지+버튼) | 낮음 (시간+준비물 1줄) |
| **CTA** | 토글 버튼 | 직진 버튼 "가이드 시작하기" |
| **배경** | Card variant="info" | 미묘한 그라데이션 |

### 5.4 가이드 카드

| 항목 | 현재 | 변경 |
|------|------|------|
| **아이콘** | 없음 | 좌측 카테고리 아이콘 배지 |
| **카테고리** | 우측 상단 배지 | 하단 메타 정보로 이동 |
| **클릭 힌트** | .click-hint 클래스 | 우측 화살표 아이콘 |
| **호버** | 없음 | 화살표 이동 + 배경 변화 |

### 5.5 상세 탭

| 항목 | 현재 | 변경 |
|------|------|------|
| **레이블** | "단계별", "체크", "FAQ", "해결" | 아이콘 + 풀 레이블 |
| **스타일** | TabsList 기본 | pill 스타일 + 선택 강조 |
| **아이콘** | 없음 | 각 탭에 아이콘 추가 |
