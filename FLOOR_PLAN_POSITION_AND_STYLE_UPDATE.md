# 약도 위치 및 스타일 수정 완료

## ✅ 완료된 작업

### 1. 7~10호 위치 수정

**파일**: `lib/constants/floorPlan.ts`

**문제점**:
- 기존 위치가 실제 약도 구조와 맞지 않음
- 7-9호가 너무 아래쪽에 배치되어 있었음

**수정 내용**:
- 7호: `x: 840, y: 220` (건물/창고 오른쪽 상단)
- 8호: `x: 840, y: 330` (건물/창고 오른쪽 중간)
- 9호: `x: 840, y: 440` (주차공간 3 오른쪽 상단)
- 10호: `x: 840, y: 550` (8호 아래, 주차공간 3 오른쪽 하단)

**새로운 배치**:
```
[건물/창고]  [7호]
[주차공간 3] [8호]
             [9호]
             [10호]
```

### 2. 건물/창고 텍스트 제거

**파일**: 
- `lib/constants/floorPlan.ts`
- `components/features/FloorPlanSVG.tsx`

**구현 내용**:
- 건물/창고의 `name` 필드를 빈 문자열로 변경
- SVG 렌더링 시 `facility.name`이 있을 때만 텍스트 표시
- 건물/창고는 시각적으로만 표시 (회색 박스)

**변경 사항**:
```typescript
// 기존
{
  id: 'warehouse',
  name: '건물/창고',
  ...
}

// 변경
{
  id: 'warehouse',
  name: '', // 텍스트 노출하지 않음
  ...
}
```

### 3. 주차공간 별도 색상 구분

**파일**: `components/features/FloorPlanSVG.tsx`

**구현 내용**:
- 주차공간 색상을 파란색 계열로 변경
- 배경: `#dbeafe` (밝은 파란색)
- 테두리: `#3b82f6` (진한 파란색)
- 다른 시설과 명확히 구분 가능

**색상 구분**:
- **주차공간**: 파란색 계열 (`#dbeafe` 배경, `#3b82f6` 테두리)
- **관리동/건물**: 회색 계열 (`#e5e7eb` 배경, `#6b7280` 테두리)
- **카페**: 노란색 계열 (`#fef3c7` 배경, `#f59e0b` 테두리)
