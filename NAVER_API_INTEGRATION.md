# 네이버 예약 API 직접 연동 가이드

## 📋 개요

네이버 예약 API를 직접 연동하여 실시간으로 예약 정보를 받아오는 방법입니다.

## ⚠️ 중요 사항

### 네이버 예약 API 현황

**현재 네이버 예약은 공식 API를 제공하지 않습니다.**

네이버 예약은 다음과 같은 제한이 있습니다:
- 공식 REST API 없음
- 웹훅(Webhook) 지원 없음
- 실시간 데이터 연동 불가

### 대안 방법

#### 방법 1: 이메일 트래킹 (현재 권장 방법)
- ✅ 구현 가능
- ✅ 안정적
- ✅ 즉시 사용 가능
- 네이버 예약 완료 시 발송되는 이메일을 감지

#### 방법 2: 웹 스크래핑 (비권장)
- ⚠️ 네이버 이용약관 위반 가능
- ⚠️ 불안정 (웹사이트 변경 시 작동 중단)
- ⚠️ 법적 리스크

#### 방법 3: 네이버 예약 파트너 프로그램
- 네이버 예약 파트너 프로그램 가입 필요
- API 제공 여부 확인 필요
- [네이버 예약 파트너 센터](https://partner.booking.naver.com/) 문의
