# UI 컴포넌트 인벤토리

## 📋 개요

이 문서는 프로젝트에서 사용 중인 모든 UI 컴포넌트를 정리합니다.

---

## ✅ 사용 중인 UI 컴포넌트

### Shadcn UI 컴포넌트 (`components/ui/`)

#### 1. Button (`button.tsx`)
- **사용 위치**: 모든 페이지 및 컴포넌트
- **용도**: 버튼 액션
- **상태**: ✅ 사용 중

#### 2. Card (`card.tsx`)
- **사용 위치**: 
  - 관리자 페이지 (대시보드, 예약 목록, 방 관리, 주문 관리)
  - 고객 페이지 (홈, 안내, 주문, 체크인/체크아웃, 도움말)
  - 기능 컴포넌트 (WiFi 카드, 시간 카드, 일몰 위젯 등)
- **용도**: 콘텐츠 카드 컨테이너
- **상태**: ✅ 사용 중

#### 3. Input (`input.tsx`)
- **사용 위치**:
  - 관리자 로그인 페이지
  - 예약 상세 페이지 (전화번호 입력)
  - 방 관리 페이지 (방 정보 입력)
  - 필터 컴포넌트 (검색, 날짜 입력)
- **용도**: 텍스트 입력 필드
- **상태**: ✅ 사용 중

#### 4. Label (`label.tsx`)
- **사용 위치**:
  - 관리자 로그인 페이지
  - 예약 상세 페이지
  - 방 관리 페이지
  - 필터 컴포넌트
- **용도**: 입력 필드 레이블
- **상태**: ✅ 사용 중

#### 5. Badge (`badge.tsx`)
- **사용 위치**:
  - 예약 목록 (상태 표시)
  - 주문 목록 (상태 표시)
  - 방 관리 (상태 표시)
- **용도**: 상태 표시 배지
- **상태**: ✅ 사용 중

#### 6. Select (`select.tsx`)
- **사용 위치**:
  - 예약 상세 페이지 (방 선택)
  - 필터 컴포넌트 (상태 선택)
- **용도**: 드롭다운 선택
- **상태**: ✅ 사용 중

#### 7. Skeleton (`skeleton.tsx`)
- **사용 위치**:
  - 관리자 대시보드 (로딩 상태)
  - 예약 목록 (로딩 상태)
  - 일몰 위젯 (로딩 상태)
- **용도**: 로딩 상태 표시
- **상태**: ✅ 사용 중

#### 8. Dialog (`dialog.tsx`)
- **사용 위치**:
  - 방 관리 페이지 (방 추가/수정 다이얼로그)
  - 주문 관리 페이지 (주문 상세 다이얼로그)
  - 주문 폼 (주문 확인 다이얼로그)
- **용도**: 모달 다이얼로그
- **상태**: ✅ 사용 중

#### 9. Accordion (`accordion.tsx`)
- **사용 위치**:
  - 고객 안내 페이지 (가이드 아코디언)
- **용도**: 접을 수 있는 콘텐츠 섹션
- **상태**: ✅ 사용 중

#### 10. Toast (`toast.tsx`, `toaster.tsx`, `use-toast.ts`)
- **사용 위치**: 모든 페이지 및 컴포넌트
- **용도**: 알림 메시지 표시
- **상태**: ✅ 사용 중

---

## 📊 컴포넌트 사용 통계

### 가장 많이 사용되는 컴포넌트
1. **Button** - 모든 페이지에서 사용
2. **Card** - 대부분의 페이지에서 사용
3. **Input** - 폼 및 필터에서 사용
4. **Badge** - 상태 표시에 사용

### 페이지별 사용 컴포넌트

#### 관리자 페이지
- Button, Card, Input, Label, Badge, Select, Skeleton, Dialog, Toast

#### 고객 페이지
- Button, Card, Input, Label, Accordion, Toast

#### 필터 컴포넌트
- Card, Input, Label, Select, Button
