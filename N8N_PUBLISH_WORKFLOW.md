# n8n 워크플로우 Publish 가이드

## 🔍 n8n 워크플로우 활성화 방법

**n8n에서 워크플로우를 활성화하는 방법:**
- **"Publish" 버튼**을 클릭하면 워크플로우가 활성화됨
- "Active" 토글은 최신 버전에서 "Publish" 버튼으로 대체됨
- 무료 플랜(trial 포함)에서도 워크플로우 활성화 가능

---

## ✅ 워크플로우 활성화 단계

### 방법 1: Publish 버튼 사용 (권장)

**워크플로우 편집기에서:**

1. **n8n 워크플로우 편집기** 상단 확인
2. **"Publish" 버튼** 클릭 (우측 상단)
3. 워크플로우가 활성화됨
4. "Saved" 텍스트가 "Published"로 변경됨

**활성화 확인:**
- "Publish" 버튼이 "Unpublish"로 변경됨
- 또는 "Published" 상태 표시
- 워크플로우가 자동으로 실행 시작

---

### 방법 2: 워크플로우 목록에서 활성화

**n8n 대시보드 → Workflows:**

1. **"Workflows"** 메뉴 클릭
2. 워크플로우 목록에서 **"네이버 예약 → 카카오톡 발송"** 찾기
3. 워크플로우 카드에서 **"Publish" 버튼** 또는 **토글** 클릭
4. 워크플로우가 활성화됨

---

## 🔧 Publish vs Active

**n8n 버전별 차이:**

**이전 버전:**
- "Active" 토글로 활성화/비활성화

**최신 버전:**
- "Publish" 버튼으로 활성화
- "Unpublish" 버튼으로 비활성화

**기능:**
- "Publish" = 워크플로우 활성화
- "Unpublish" = 워크플로우 비활성화

---

## 📋 워크플로우 상태 확인

**활성화된 워크플로우:**
- "Publish" 버튼이 "Unpublish"로 변경
- 또는 "Published" 상태 표시
- 워크플로우 카드에 "Active" 배지 표시

**비활성화된 워크플로우:**
- "Publish" 버튼 표시
- 또는 "Unpublished" 상태 표시
- 워크플로우 카드에 "Inactive" 배지 표시

---

## 🚀 워크플로우 활성화 후 동작

**Gmail Trigger 동작:**

**활성화된 워크플로우:**
- 설정한 **Poll Times** (예: "Every Minute")마다 Gmail 확인
- 새로운 예약 이메일이 오면 자동으로 처리
- 각 이메일을 하나씩 처리

**비활성화된 워크플로우:**
- Gmail 확인하지 않음
- 이메일이 와도 처리하지 않음

---

## 🔍 실행 로그 확인

**n8n 대시보드 → 워크플로우 → "Executions" 탭:**

1. **"Executions" 탭** 클릭
2. 워크플로우 실행 기록 확인
3. 각 실행의 상태 확인:
   - ✅ **Success**: 성공
   - ❌ **Error**: 실패
   - ⏸️ **Running**: 실행 중

## 🐛 문제 해결

### 문제 1: Publish 버튼이 작동하지 않음

**증상:**
- "Publish" 버튼을 클릭해도 활성화되지 않음

**해결:**
1. 워크플로우 저장 확인 ("Saved" 표시)
2. 모든 노드가 올바르게 설정되었는지 확인
3. 에러가 있는 노드가 없는지 확인
4. 브라우저 새로고침 후 다시 시도

---

### 문제 2: 워크플로우가 실행되지 않음

**증상:**
- "Published" 상태이지만 실행되지 않음

**해결:**
1. Gmail Trigger 설정 확인
2. Gmail 인증 확인
3. Poll Times 확인
4. Executions 탭에서 에러 확인

---

### 문제 3: 실행 횟수 제한

**증상:**
- "Execution limit reached" 메시지

**해결:**
1. Trial 플랜의 실행 횟수 확인
2. 유료 플랜으로 업그레이드 고려
3. 또는 다음 달까지 대기

---

## 🔍 워크플로우 상태 확인 방법

**워크플로우 편집기 상단:**
- "Publish" 버튼 → 비활성화 상태
- "Unpublish" 버튼 → 활성화 상태
- "Published" 텍스트 → 활성화 상태

**워크플로우 목록:**
- 워크플로우 카드에 "Active" 배지 → 활성화
- 워크플로우 카드에 "Inactive" 배지 → 비활성화
