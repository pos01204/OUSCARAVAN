# 카카오 Redirect URI 설정 가이드

## 📍 현재 위치

**현재 페이지**: 웹훅 설정 페이지  
**필요한 페이지**: 카카오 로그인 설정 페이지

---

## 🎯 Redirect URI 설정 위치

Redirect URI는 **"카카오 로그인"** 설정 페이지에 있습니다.

### 이동 방법:

1. 왼쪽 사이드바에서 **"제품 설정"** 섹션 확인
2. **"카카오 로그인"** 클릭
3. 카카오 로그인 설정 페이지로 이동

---

## 📋 단계별 설정

### 1단계: 카카오 로그인 페이지로 이동

#### 현재 화면에서:

1. 왼쪽 사이드바의 **"제품 설정"** 섹션 확인
2. **"카카오 로그인"** 메뉴 클릭
3. 카카오 로그인 설정 페이지로 이동

### 2단계: 카카오 로그인 활성화

카카오 로그인 설정 페이지에서:

1. **"활성화 설정"** 토글을 **ON**으로 변경
2. **저장** 클릭

### 3단계: Redirect URI 추가

카카오 로그인 설정 페이지에서:

1. **"Redirect URI"** 섹션 찾기
2. **"URI 추가"** 또는 **"+ 추가"** 버튼 클릭
3. 다음 URI 입력:
   ```
   https://ouscaravan.com/auth/kakao/callback
   ```
   (또는 실제 배포된 도메인)
4. **저장** 또는 **추가** 클릭

### 4단계: 동의항목 설정

1. **"동의항목"** 탭 클릭
2. 필요한 동의항목 활성화:
   - **전화번호**: 선택 (카카오톡 메시지 발송에 필요)
   - **카카오톡 메시지 전송**: 선택 (필수)
3. **저장** 클릭

---

## 🔍 카카오 로그인 설정 페이지 구조

카카오 로그인 설정 페이지에는 다음 섹션들이 있습니다:

1. **활성화 설정**
   - 토글 스위치로 ON/OFF

2. **Redirect URI**
   - URI 목록
   - URI 추가 버튼
   - **여기에 Redirect URI를 추가합니다!**

3. **동의항목**
   - 필수/선택 동의항목 설정

4. **고급 설정**
   - 추가 옵션들

---

## ⚠️ 중요 사항

### Redirect URI 형식

- **올바른 형식**: `https://ouscaravan.com/auth/kakao/callback`
- **잘못된 형식**: 
  - `http://` (HTTPS만 가능)
  - 끝에 슬래시 포함 (`/callback/`)
  - 포트 번호 포함

### 여러 Redirect URI 추가 가능

개발/프로덕션 환경을 위해 여러 URI를 추가할 수 있습니다:
- `https://ouscaravan.com/auth/kakao/callback` (프로덕션)
- `http://localhost:3000/auth/kakao/callback` (개발용, 선택사항)

## 📋 체크리스트

---

## 🆘 Redirect URI를 찾을 수 없는 경우

### 확인 사항:

1. **카카오 로그인이 활성화되어 있는지**
   - 활성화 설정이 ON이어야 Redirect URI 섹션이 보입니다

2. **올바른 페이지에 있는지**
   - "앱 설정" → "앱" → "웹훅" (현재 위치) ❌
   - "제품 설정" → "카카오 로그인" (필요한 위치) ✅

3. **권한 확인**
   - Owner 권한이 있는지 확인
