# 카카오톡 메시지 활성화 가이드 (현재 화면 기준)

## 📍 현재 위치

**현재 페이지**: 추가 기능 신청 페이지  
**필요한 페이지**: 카카오톡 메시지 설정 페이지

---

## 🎯 카카오톡 메시지 활성화 위치

카카오톡 메시지 API는 **"제품 설정"** → **"카카오톡 메시지"**에서 활성화합니다.

### 현재 화면에서 이동:

1. 왼쪽 사이드바의 **"제품 설정"** 섹션 확인
2. **"카카오톡 메시지"** 메뉴 클릭
3. 카카오톡 메시지 설정 페이지로 이동

---

## 📋 단계별 설정

### 1단계: 카카오톡 메시지 페이지로 이동

#### 현재 화면에서:

1. 왼쪽 사이드바의 **"제품 설정"** 섹션 확인
   - 현재 "제품 설정"이 확장되어 보일 수 있음
2. **"카카오톡 메시지"** 클릭
3. 카카오톡 메시지 설정 페이지로 이동

### 2단계: 친구톡 활성화

카카오톡 메시지 설정 페이지에서:

1. **"친구톡"** 섹션 확인
2. **"활성화 설정"** 토글을 **ON**으로 변경
3. **저장** 클릭

### 3단계: 추가 기능 신청 (필요한 경우)

현재 보고 있는 "추가 기능 신청" 페이지에서:

#### 카카오톡 친구 목록/메시지 신청

1. **"카카오톡 친구 목록/메시지"** 행 찾기
2. **"신청"** 버튼 클릭
3. 신청 사유 입력:
   ```
   네이버 예약 완료 시 게스트에게 카카오톡 메시지로 고유 링크를 발송하기 위함
   ```
4. **제출** 클릭
5. 심사 승인 대기 (보통 몇 시간~1일 소요)

**참고**: 
- 이 신청은 선택사항일 수 있습니다
- 먼저 "제품 설정" → "카카오톡 메시지"에서 친구톡을 활성화해보세요
- 활성화가 안 되면 이 신청이 필요할 수 있습니다

---

## 🔍 두 가지 경로

### 경로 1: 제품 설정에서 직접 활성화 (먼저 시도)

1. **"제품 설정"** → **"카카오톡 메시지"** 클릭
2. 친구톡 활성화 시도
3. 활성화가 되면 완료!

### 경로 2: 추가 기능 신청 후 활성화 (필요한 경우)

1. 현재 페이지에서 **"카카오톡 친구 목록/메시지"** 신청
2. 심사 승인 대기
3. 승인 후 **"제품 설정"** → **"카카오톡 메시지"**에서 활성화

## 🆘 문제 해결

### "카카오톡 메시지" 메뉴가 보이지 않는 경우

**확인 사항:**
- 카카오 로그인이 활성화되어 있는지
- 앱 상태가 정상인지

### 친구톡 활성화가 안 되는 경우

**해결 방법:**
1. "추가 기능 신청"에서 "카카오톡 친구 목록/메시지" 신청
2. 심사 승인 대기
3. 승인 후 다시 활성화 시도
