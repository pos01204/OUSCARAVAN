# 카카오톡 친구 목록/메시지 추가 기능 신청 사유 및 활용 시나리오

## 신청 사유

OUSCARAVAN은 프리미엄 캠핑장/글램핑 시설로, 게스트에게 최고의 호스피탈리티 경험을 제공하기 위해 디지털 컨시어지 서비스를 운영하고 있습니다. 카카오톡 메시지를 통한 자동화된 예약 관리 및 안내 서비스를 제공하여 게스트의 편의성을 극대화하고, 관리자의 업무 효율성을 향상시키고자 합니다.

---

## 활용 시나리오

### 1. 카라반 예약 시

**시나리오:**
네이버 예약을 통해 예약이 완료되면, 자동으로 게스트에게 카카오톡 메시지를 발송하여 예약 확인 및 컨시어지 서비스 안내를 제공합니다.

**발송 메시지 예시:**
```
[홍길동]님, OUSCARAVAN 예약이 완료되었습니다!

📅 체크인: 2026-01-15 15:00
📅 체크아웃: 2026-01-17 11:00
🏠 객실: Airstream1
💰 결제금액: 180,000원

아래 링크를 클릭하여 컨시어지 서비스를 이용하세요.
[컨시어지 서비스 이용하기] 버튼
```

**기대 효과:**
- 게스트가 즉시 예약 정보를 확인할 수 있어 안심감 제공
- 컨시어지 서비스 링크를 통한 사전 정보 제공으로 체크인 시간 단축
- 반복적인 전화 문의 감소로 관리자 업무 효율성 향상

---

### 2. 카라반 체크인 당일

**시나리오:**
체크인 당일 오전, 게스트에게 카카오톡 메시지를 발송하여 체크인 안내 및 웹 앱 링크를 제공합니다. 이를 통해 WiFi 정보, 객실 가이드, 주변 정보 등을 사전에 확인할 수 있도록 합니다.

**발송 메시지 예시:**
```
[홍길동]님, 오늘 OUSCARAVAN에 체크인하시는 날입니다!

⏰ 체크인 시간: 오후 3시
📍 위치: 강원도 속초시...

체크인 전에 아래 링크에서 필요한 정보를 확인하세요:
- WiFi 비밀번호
- 객실 사용 가이드
- 불멍/바베큐 주문
- 주변 맛집 정보

[컨시어지 서비스 이용하기] 버튼
```

**기대 효과:**
- 체크인 전 필요한 정보를 사전에 제공하여 체크인 프로세스 원활화
- 현장에서의 반복적인 안내 업무 감소
- 게스트의 자율적인 정보 접근으로 만족도 향상

---

### 3. 카라반 체크아웃 당일

**시나리오:**
체크아웃 당일 오전, 게스트에게 카카오톡 메시지를 발송하여 체크아웃 시간 안내 및 체크아웃 체크리스트를 제공합니다. 또한 자동 체크아웃 기능을 통해 게스트가 직접 체크아웃을 완료할 수 있도록 합니다.

**발송 메시지 예시:**
```
[홍길동]님, 오늘 체크아웃하시는 날입니다!

⏰ 체크아웃 시간: 오전 11시

체크아웃 전 확인사항:
✓ 개인 소지품 확인
✓ 객실 정리 상태 확인
✓ 키 반납 (또는 자동 체크아웃)

아래 링크에서 자동 체크아웃을 진행하세요.
[체크아웃하기] 버튼
```

**기대 효과:**
- 체크아웃 시간 준수율 향상
- 체크아웃 프로세스 자동화로 관리자 업무 부담 감소
- 게스트의 편의성 향상 (현장 방문 없이 체크아웃 가능)

---

## 기술적 구현 방식

### 자동화 시스템
- **n8n 워크플로우**: 네이버 예약 이메일 감지 및 자동 처리
- **Gmail API**: 예약 완료/취소 이메일 모니터링
- **카카오톡 메시지 API**: 친구 목록 조회 및 메시지 발송
- **웹 앱 연동**: 고유 링크를 통한 개인화된 컨시어지 서비스 제공

### 데이터 흐름
```
[네이버 예약 완료]
  ↓
[예약 완료 이메일 발송]
  ↓
[n8n Gmail Trigger 감지]
  ↓
[게스트 정보 추출 및 저장]
  ↓
[카카오톡 친구 목록 조회]
  ↓
[카카오톡 메시지 발송 (고유 링크 포함)]
  ↓
[게스트가 링크 클릭]
  ↓
[웹 앱 접속 (자동 로그인)]
  ↓
[컨시어지 서비스 이용]
```

## 비즈니스 목적

### 1. 게스트 경험 향상
- 예약부터 체크아웃까지 전 과정의 자동화된 안내
- 필요한 정보에 즉시 접근 가능
- 프리미엄 서비스 경험 제공

### 2. 운영 효율성 향상
- 반복적인 전화 문의 감소 (예상 70% 감소)
- 체크인/체크아웃 프로세스 자동화
- 관리자 업무 시간 절감 (게스트당 평균 15분 → 3분)

### 3. 서비스 품질 향상
- 일관된 안내 서비스 제공
- 실시간 정보 업데이트
- 오류 감소 (수기 기록 및 전달 과정에서 발생하는 실수 방지)

---

## 개인정보 보호

- 카카오톡 메시지 발송은 게스트의 명시적 동의 하에 진행됩니다.
- 예약 완료 시 제공된 연락처 정보만 사용합니다.
- 개인정보는 암호화되어 저장되며, 체크아웃 후 일정 기간 후 자동 삭제됩니다.
- 게스트는 언제든지 메시지 수신을 거부할 수 있습니다.

카카오톡 친구 목록/메시지 기능을 활용하여 OUSCARAVAN은 게스트에게 더 나은 서비스를 제공하고, 운영 효율성을 크게 향상시킬 수 있습니다. 이를 통해 프리미엄 호스피탈리티 브랜드로서의 경쟁력을 강화하고, 게스트 만족도를 높이고자 합니다.
